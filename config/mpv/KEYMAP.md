# mpv 快捷键速查表

基于当前目录下的 `input.conf` 整理。

## 最常用

| 按键               | 功能                |
| ------------------ | ------------------- |
| `Space` / 鼠标左键 | 播放 / 暂停         |
| `q`                | 退出                |
| `Q`                | 退出并保存进度      |
| `f` / `Enter`      | 切换全屏            |
| `m`                | 静音                |
| `Right` / `Left`   | 快进 / 后退 5 秒    |
| `Up` / `Down`      | 快进 / 后退 60 秒   |
| `.` / `,`          | 前进一帧 / 后退一帧 |
| `0` / `9`          | 音量 +5 / -5        |
| `]` / `[`          | 加速 / 减速 0.1     |
| `Backspace`        | 恢复 1.0x           |
| `j`                | 切换字幕轨          |
| `v`                | 显示 / 隐藏字幕     |
| `y`                | 切换音轨            |
| `s`                | 截图（含字幕）      |
| `S`                | 截图（仅视频）      |
| `i`                | 显示统计信息        |

## 播放与退出

| 按键                           | 功能           |
| ------------------------------ | -------------- |
| `Space` / 鼠标左键             | 播放 / 暂停    |
| `PlayPause` / `Play` / `Pause` | 播放 / 暂停    |
| `q`                            | 退出           |
| `Q`                            | 退出并保存进度 |
| `Esc`                          | 退出全屏       |
| `f` / `Enter`                  | 切换全屏       |
| `m`                            | 静音           |
| `Power` / `Stop`               | 退出           |

## 跳转与逐帧

| 按键                         | 功能                 |
| ---------------------------- | -------------------- |
| `Right` / `Left`             | 快进 / 后退 5 秒     |
| `Up` / `Down`                | 快进 / 后退 60 秒    |
| `Shift+Right` / `Shift+Left` | 精确快进 / 后退 1 秒 |
| `Forward` / `Rewind`         | 快进 / 后退 30 秒    |
| `.`                          | 前进一帧             |
| `,`                          | 后退一帧             |

## 音量与速度

| 按键            | 功能                 |
| --------------- | -------------------- |
| 鼠标滚轮上 / 下 | 音量 +2 / -2         |
| `0` / `9`       | 音量 +5 / -5         |
| `]` / `[`       | 播放速度 +0.1 / -0.1 |
| `Backspace`     | 恢复到 1.0x          |

## 播放列表与章节

| 按键            | 功能                    |
| --------------- | ----------------------- |
| `PgUp` / `PgDn` | 下一章 / 上一章         |
| `>` / `<`       | 下一项 / 上一项播放列表 |
| `Next` / `Prev` | 下一项 / 上一项播放列表 |

## 字幕

| 按键              | 功能                       |
| ----------------- | -------------------------- |
| `j`               | 切换字幕轨                 |
| `v`               | 显示 / 隐藏字幕            |
| `z` / `x`         | 字幕提前 / 延后 0.1 秒     |
| `r` / `t`         | 上移 / 下移字幕            |
| `Shift+Backspace` | 重置字幕位置 / 缩放 / 延迟 |
| `F11`             | 打开字幕轨列表             |
| `Ctrl+f`          | 打开字幕行列表             |

## 音轨

| 按键                | 功能                   |
| ------------------- | ---------------------- |
| `y`                 | 切换音轨               |
| `Ctrl+,` / `Ctrl+.` | 音频提前 / 延后 0.1 秒 |
| `;`                 | 重置音频延迟           |
| `F10`               | 打开音轨列表           |
| `F6`                | 打开音频输出设备列表   |

## 画面调整

| 按键             | 功能             |
| ---------------- | ---------------- |
| `A`              | 切换画面比例     |
| `d`              | 开关隔行扫描处理 |
| `D`              | 开关 deband      |
| `1` / `2`        | 对比度 -1 / +1   |
| `3` / `4`        | 亮度 -1 / +1     |
| `5` / `6`        | Gamma -1 / +1    |
| `7` / `8`        | 饱和度 -1 / +1   |
| `Ctrl+Backspace` | 重置画面参数     |
| `C`              | 切换动态去黑边   |

## Shader / 画质

| 按键     | 功能                         |
| -------- | ---------------------------- |
| `U`      | 切换 `adaptive-sharpen.glsl` |
| `Ctrl+U` | 清空当前 shader              |
| `Ctrl+F` | 打开视频格式 / 清晰度菜单    |
| `Alt+F`  | 打开音频格式菜单             |
| `Alt+Y`  | 打开 YouTube 下载菜单        |

## 截图与信息

| 按键     | 功能             |
| -------- | ---------------- |
| `s`      | 截图（含字幕）   |
| `S`      | 截图（仅视频）   |
| `Ctrl+s` | 截取窗口         |
| `i`      | 显示统计信息     |
| `I`      | 切换统计信息常驻 |
| `O`      | 切换 OSD 级别    |

| 按键     | 功能              |

## 动图与切片

| 按键     | 功能             |
| -------- | ---------------- |
| `w`      | 记录动图开始时间 |
| `W`      | 记录动图结束时间 |
| `Ctrl+w` | 生成动图         |
| `Ctrl+W` | 生成带字幕动图   |

| `Alt+a`  | 导出音频切片     |

## 复制信息到剪贴板

| 按键         | 功能             |
| ------------ | ---------------- |
| `Ctrl+Alt+t` | 复制当前播放时间 |
| `Ctrl+Alt+p` | 复制当前文件路径 |
| `Ctrl+Alt+s` | 复制当前字幕文本 |

## 备注

- 本表仅整理当前配置中已绑定的按键。
- 部分功能依赖脚本，如 `uosc`、`file_browser`、`recentmenu`、`simplebookmark`、`quality_menu`、`youtube_download`、`dynamic_crop`、`slicing_copy`、`mpv_animated` 等。
- 如果脚本不存在，对应按键可能无效果。
