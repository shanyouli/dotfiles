# shared → tui → modern

**源文件**: `modules/shared/tui/modern.nix`  
**选项前缀**: `config.modules.shared.tui.modern`

## 概述

安装一些现代工具
现代 cli 工具来源:
see @https://github.com/hugsy/modern
see @https://github.com/ibraheemdev/modern-unix

## 选项

| 选项 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `enable` | bool | Whether modern tools are used, eg:duf,exa .etc | 是否启用 |
