# shared → tui → media → music → mpd → rmpc

**源文件**: `modules/shared/tui/media/music/mpd/rmpc.nix`  
**选项前缀**: `config.modules.media.music.mpd`

## 概述

使用 rust 编写的 mpd 客户端
see @https://mierak.github.io/rmpc/
only support linux

## 选项

| 选项 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `enable` | bool | Whether to use rmpc | 是否启用 |
